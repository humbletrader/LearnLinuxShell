# Bash job management : 
* CTRL+C - kills the job
* CTRL+Z - stops a running job 
* jobs - listing of stopped programs 
* fg - brings a stopped program to foreground ( starts again)
* bg - background 
* & - starts a program directly into background
```bash
htop

CTRL-Z

jobs
[1]+ Stopped htop

fg 1
```
